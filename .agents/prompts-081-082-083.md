# Prompts para Codex (081), GitHub Copilot (082) y Antigravity (083)

Cada agente trabaja **en su propia rama**. Antes de empezar: **haz pull de main** (`git pull origin main` o `git checkout main && git pull origin main`). Luego crea tu rama desde main actualizado.

```
Tarea 081: capitalizar trabajo en ramas. Sigue EXACTAMENTE .agents/tasks/2026-03-05-081-r16-codex-capitalizar-ramas.md.

IMPORTANTE: Trabaja SOLO en la rama codex/081-capitalizar-ramas.

Haz solo:
1. Listar ramas remotas con commits no en main; anotar rama y resumen.
2. Crear docs/informe-ramas-pendientes.md con tabla: rama | PR | resumen 1 línea | recomendación.
3. Recuperar 1–2 docs seguros (solo docs, sin código) por cherry-pick o copia a tu rama.
4. Abrir PR desde codex/081-capitalizar-ramas a main.

No refactorices, no toques tests ni dependencias. Solo inventario + recuperación de documentos.
```

## Para Antigravity — Tarea 083

```
Tarea 083: análisis profundo de contenido útil perdido en PRs/commits cerrados para capitalizar todo el trabajo. Sigue EXACTAMENTE .agents/tasks/2026-03-05-083-r16-antigravity-analisis-contenido-perdido.md.

3. Crear docs/analisis-contenido-perdido-r16.md con: resumen ejecutivo, tabla rama | PR | archivos | valoración | acción recomendada, y sección "Prioridad de recuperación" (top 5–10 items a recuperar).
4. Abrir PR desde antigravity/083-analisis-contenido-perdido a main (solo el informe).

No mergear nada ni tocar código de aplicación. Solo análisis y documento.
```
