## Errores Resueltos

- Gateway token missing → configurar y reiniciar systemd
- WORKER_TOKEN not set → definir en env antes de iniciar
- 401 unauthorized → formato correcto de Authorization header
- bash "event not found" → comillas simples para tokens con `!`
- Telegram 409 conflict → solo una instancia del bot
- Plugins install E404 → providers son nativos, usar auth setup

## Decisiones

- Telegram solo en VPS (no en Windows, para evitar 409)
- Anthropic como fallback (no default)
- Antigravity mantenido por decisión del usuario (riesgo documentado)
- SSH tunnel preferido sobre Tailscale para Control UI
