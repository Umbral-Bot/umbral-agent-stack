# 31 — Prompt para Enlace: análisis de oportunidades y perfil

> Prompt para pedir a Enlace (Notion) que analice proyectos para retomar y haga un análisis del perfil de David. Usar cuando David pida directo a Enlace o cuando Rick delegue en Enlace.

---

## Prompt para Enlace

Copia y pega este prompt (o una variante) como comentario en la Control Room o en una página del alcance de Enlace, dirigiéndote a @Enlace:

```
Hola @Enlace,

Revisa todas las páginas del workspace a las que tienes acceso, busca oportunidades para retomar proyectos que estén pausados o con poco avance, y haz un análisis comprometido de mi perfil (David Moreira) y de lo que hago en Umbral BIM / Umbral Arquitectura.

Incluye:
1. Proyectos que conviene retomar (prioridad, estado, siguiente paso).
2. Análisis de mi perfil profesional y cómo encaja con los proyectos activos.
3. Recomendaciones concretas de acciones para las próximas 2 semanas.

Deja el resultado en Bandeja Puente (o en esta página) para que Rick o yo podamos revisarlo.
```

---

## Uso vía Rick (delegación)

Si quieres que Rick se lo pida a Enlace por ti, comenta en Control Room:

```
Rick: Pídele a Enlace lo siguiente: [copia aquí el prompt de arriba, o una versión acortada]
```

Ejemplo acortado:

```
Rick: Pídele a Enlace que revise todas las páginas del workspace, busque oportunidades para retomar proyectos, haga un análisis de mi perfil (David Moreira) y de lo que hago, y deje el resultado en Bandeja Puente.
```
