# MyClaw

> MyClaw is a managed hosting platform for OpenClaw (formerly Clawdbot) — an open-source AI computer-use assistant. MyClaw provides always-on cloud instances with subscription plans starting at $19/month, so users can run their personal AI assistant 24/7 without any DevOps.

MyClaw handles VM provisioning, automatic updates, security, and backups. Users interact with their OpenClaw instance through a web-based terminal and chat interface. The platform also offers API credits for programmatic access, a referral program, and an affiliate partner program.

## Pricing

- [Pricing](https://myclaw.ai/pricing): Three subscription tiers — Lite ($19/mo, 2 vCPU, 4GB RAM), Pro ($39/mo, 4 vCPU, 8GB RAM), Max ($79/mo, 8 vCPU, 16GB RAM). Annual plans available at ~15% discount.
- [Compare](https://myclaw.ai/compare): Comparison of Local vs VPS vs MyClaw hosting — covers setup time, technical skill, uptime, security, maintenance, and cost.
