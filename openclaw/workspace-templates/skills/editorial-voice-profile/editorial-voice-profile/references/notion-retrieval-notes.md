# Notion retrieval notes

Usar esta referencia cuando el perfil editorial dependa de contenido recuperado desde Notion dentro de ChatGPT.

## Reglas practicas

- tratar la fuente como de solo lectura
- asumir que puede haber desfase entre el contenido mas reciente y el contenido disponible en una app con sincronizacion
- no asumir cobertura total del workspace si el usuario no dio paginas concretas o texto pegado
- no depender de imagenes, videos o otros elementos no textuales para inferencias finas de voz
- si el contenido esta en tablas o bases de datos, describirlo de forma amplia como database o data source y enfocarse en las filas o paginas realmente visibles

## Como actuar cuando la cobertura es parcial

1. decir que la propuesta es provisional
2. indicar que canales o periodos quedaron poco representados
3. bajar el nivel de confianza de los campos afectados
4. pedir, si hace falta, paginas concretas o texto pegado para afinar
5. no convertir silencios de evidencia en reglas permanentes

## Senales de que hace falta mas material

- casi todo el corpus viene de un solo canal
- hay muchos fragmentos y pocas piezas completas
- el usuario pide una voz muy precisa pero solo comparte notas breves
- la estrategia de cta cambia mucho entre piezas y no hay suficiente contexto para explicarla
- los referentes estan mejor representados que la voz propia del autor
