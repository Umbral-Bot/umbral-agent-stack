# Evidence Thresholds

## Objetivo

Decidir cuanto se puede afirmar segun el tipo de referencia, la evidencia reunida y el nivel de cierre pretendido.

No todas las referencias piden el mismo rigor, pero ninguna con URL o referente concreto deberia cerrarse sin evidencia real si habia tools disponibles.

## Niveles de cierre

### 1. Senal parcial

Usar cuando:
- solo hay una pieza aislada
- la evidencia es limitada
- el acceso es parcial o el usuario solo pego un fragmento

Permite:
- describir lo observado
- rescatar una idea puntual
- proponer una lectura inicial

No permite:
- afirmar estrategia completa
- afirmar funnel
- afirmar arquitectura de activos

### 2. Lectura aplicada

Usar cuando:
- se observo al menos una fuente real con suficiente contexto
- ya se puede traducir a una adaptacion razonable

Permite:
- extraer framing, CTA, estructura o estilo pedagogico
- decidir un posible fit con un proyecto o sistema
- proponer un siguiente paso

No permite:
- vender como hecho lo no observado
- afirmar backend, secuencia o continuidad no verificadas

### 3. Decision de integracion

Usar cuando:
- existe evidencia real suficiente
- el hallazgo fue traducido a una recomendacion concreta
- ya se decidio donde encaja o por que no encaja

Permite:
- recomendar persistencia
- abrir backlog, entregable o update de proyecto

### 4. Referencia integrada

Usar cuando:
- ademas del analisis, se genero artefacto o persistencia real
- existe update de proyecto, entregable o rastro de implementacion proporcional al caso

Permite:
- tratar la referencia como trabajo operativo concluido

## Minimo por tipo de pedido

### A. "Mira esta publicacion y saca lo valioso"

Minimo:
- inspeccion real del post o del material provisto
- evidencia observada separada de inferencia
- 1 a 3 aprendizajes aplicables
- decision de si requiere persistencia

### B. "Me gusta el estilo comunicativo de X"

Minimo:
- una o mas piezas observadas del referente
- separacion entre estilo visible y estrategia no verificada
- traduccion a reglas de tono, estructura o pedagogia

### C. "Revisa este referente y dime si sirve para mis proyectos"

Minimo:
- evidencia real
- evaluacion de fit con al menos un proyecto o sistema concreto
- decision explicita: sirve, sirve parcialmente o no conviene integrar

### D. Claims sobre funnel, secuencia o arquitectura de activos

Exigir uno de estos caminos:
- dos o mas fuentes independientes del mismo caso
- una fuente principal mas una capa siguiente observable

Con una sola pieza, degradar el resultado a `hipotesis de sistema`.
