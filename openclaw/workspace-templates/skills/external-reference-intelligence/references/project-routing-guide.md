# Project Routing Guide

## Objetivo

Decidir donde encaja una referencia para que no quede como insight suelto.

## Rutas frecuentes

### Proyecto Embudo Ventas

Usar cuando la referencia aporta a:
- hooks de entrada
- narrativa comercial
- CTA
- lead magnet
- puente a activo propio
- estructura de captacion o conversion

### Sistema Editorial

Usar cuando la referencia aporta a:
- backlog de temas
- framing
- sistema de publicacion
- serie de contenidos
- expansion multicanal

### Marca personal

Usar cuando la referencia aporta a:
- posicionamiento
- tono
- estilo comunicativo
- narrativa de autoridad

### Docencia o contenido educativo

Usar cuando la referencia aporta a:
- pedagogia
- claridad
- ritmo de explicacion
- secuencia didactica
- criterio para YouTube, newsletter o clases

### Automatizaciones o mejora continua

Usar cuando la referencia aporta a:
- arquitectura operativa
- criterio de automatizacion
- uso de agentes
- sistema de trabajo

## Regla

Si una referencia toca mas de un frente, elegir un proyecto primario y dejar secundarios solo como nota.

No abrir tres frentes por una sola pieza si todavia no hay evidencia para tanto.
