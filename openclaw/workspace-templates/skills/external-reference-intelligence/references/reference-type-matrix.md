# Reference Type Matrix

## Objetivo

Clasificar rapido el tipo de referencia para no aplicar el workflow equivocado.

## Tipos

### Post individual

Se usa cuando el usuario comparte una publicacion, tweet, thread, post o reel puntual.

Preguntas tipicas:
- cual es el hook
- que promesa hace
- que CTA visible tiene
- si sirve como idea comunicacional o de entrada al funnel

### Perfil o persona

Se usa cuando el usuario apunta a un referente como persona, no solo a una pieza.

Preguntas tipicas:
- como se posiciona
- que narrativa sostiene
- que estilo comunicativo tiene
- si sirve para marca personal, docencia o contenido

### Articulo o newsletter

Se usa cuando la referencia es larga y argumental.

Preguntas tipicas:
- que tesis sostiene
- como estructura la explicacion
- como combina pedagogia y venta

### Video o canal

Se usa cuando la referencia es audiovisual.

Preguntas tipicas:
- ritmo
- estructura
- pedagogia
- CTA
- continuidad entre contenido y activo propio

### Landing o funnel

Se usa cuando la referencia es claramente de captacion, conversion o nurturance.

Preguntas tipicas:
- promesa
- captacion
- friccion
- continuidad

### Referente de estilo o comunicacion

Se usa cuando el usuario dice cosas como:
- me gusta como comunica X
- me gusta el estilo de Y

En este caso no asumir funnel ni arquitectura completa si no hay evidencia.

### Caso tecnico

Se usa cuando el valor esta en el enfoque, metodo, stack o criterio operativo.

### Caso comercial

Se usa cuando el valor esta en el posicionamiento, propuesta, CTA, lead capture o venta.
