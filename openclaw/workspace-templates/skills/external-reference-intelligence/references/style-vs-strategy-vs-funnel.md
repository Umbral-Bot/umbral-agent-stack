# Style vs Strategy vs Funnel

## Objetivo

No confundir una capa con otra.

## Estilo

Senales tipicas:
- tono
- ritmo
- estructura de frases
- uso de analogias
- apertura y cierre

Con estilo visible no alcanza para afirmar sistema o funnel.

## Estrategia

Senales tipicas:
- tesis editorial repetida
- audiencia consistente
- CTA repetido
- puente entre piezas
- relacion entre activo gratuito y oferta

La estrategia exige varias piezas o una continuidad observable.

## Funnel

Senales tipicas:
- post o pieza de entrada
- perfil o landing
- captura
- siguiente paso observable
- continuidad de nurturance o venta

No afirmar funnel completo solo por una pieza o por una landing aislada.
