# OpenClaw Rick Guidelines

## Para skills de Rick

Una skill pensada para Rick debe:

- asumir uso dentro de OpenClaw
- hablar en terminos de proyecto, trazabilidad y tools reales
- preferir instrucciones operativas sobre explicación genérica
- indicar cuándo persistir en carpeta, Linear o Notion
- distinguir claramente entre benchmark, ejecución y cierre

## Evitar

- skills demasiado abstractas
- skills dependientes de proveedores concretos sin necesidad
- skills que duplican reglas ya presentes en AGENTS.md y SOUL.md
