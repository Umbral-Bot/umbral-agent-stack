# Skill Decision Framework

## Elegir una accion

### Editar

Cuando ya existe una skill y el problema es:
- falta de guardrails
- falta de ejemplos
- alcance incompleto

### Extender

Cuando la skill actual es correcta pero necesita:
- una seccion nueva
- una referencia nueva
- mejor criterio de herramientas

### Fusionar

Cuando hay dos skills con mucho solapamiento y la separacion ya no aporta claridad.

### Crear nueva

Cuando el caso:
- tiene objetivo distinto
- necesita trigger distinto
- requiere flujo propio
- o mezclarlo con otra skill empeoraria la señal
